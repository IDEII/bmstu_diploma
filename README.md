# Дипломная работа в МГТУ им. Н. Э. Баумана
## Структура проекта
### Парсер
* `antlr` - Парсер языка Cypher
### Мастер
* `main.go` - Запуск узла сети и определение его роли
* `master` - Главный узел распределенной сети
* `worker` - Подчиненные узлы сети
### Фронтенд (HTML/CSS/JavaScript)
* `app.js` - логика работы веб-страницы (написание запросов, отображение результирующих графов)
* `index.html` - Главная страница приложения
* `style.css` - Нейрослоп

## Функциональные возможности
* Написание запросов на языке Cypher
* Распределение данных между узлами
* Получение результирующих графов

## Запуск
. Установить зависимости Go
. Запустить контейнер с Neo4j
. Запустить узлы `go run main.go -cmd=worker -addr IP:PORT -id num -neo4j-port PORT`
. Запустить `go run main.go -cmd=master -cohorts=IPs -web-addr=:PORT`
. Открыть `localhost:PORT`
